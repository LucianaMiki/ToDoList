# 📝 To Do List

A simple and responsive task management application built with React Native and Expo.

---

## ✨ Features

- ✅ Create tasks
- ✏️ Edit tasks
- 🗑️ Delete tasks
- 📱 Responsive layout for mobile and web
- 🎨 Custom UI design
- 🔔 User feedback for actions

---

## 🚀 Technologies

- React Native
- Expo
- TypeScript
- Expo Router
- JSON Server

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/to-do-list.git
```

### Navigate to the project folder

```bash
cd to-do-list
```

### Install dependencies

```bash
npm install
```

### Start the mock API

```bash
npx json-server db.json
```

### Run the application

```bash
npx expo start
```
